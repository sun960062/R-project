1.欄位需要依照範例中的名稱以及順序
2.上傳檔案需為csv檔，建議下載網站提供的csv檔，直接對檔案新增資料後上傳
2.’artTitle’以及'artContent’欄位為必填的欄位
3.’artDate’的默認各式為”%Y/%m/%d %H:%M”  Ex:”2017/12/2 14:15”
—-日期格式可在網頁中自行設定，日期格式要與上傳的日期格式完全相同，否則會出錯 
—-已默認日期格式為例，會造成錯誤的情況:
—-”2017/12/2  14:15”（中間有兩個空白)，”2017-12-2 14:15”（日期間隔符號不同）
—-”2017/12/2”（缺少時間）
4.’artDate’可不填，文章時間會以上傳時間取代
5.’artUrl’可不填，若不填則沒有此欄位
6.上傳檔案需要符合第二項以及第三項的要求，否則無法上傳
